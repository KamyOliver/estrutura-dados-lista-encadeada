# estrutura-dados-lista-encadeada
Simulador de fila de atendimento em C utilizando listas encadeadas. Implementa gerenciamento de senhas comuns e VIP, controle de fila, remoção de atendimentos e consulta de tamanho da fila. O projeto foi implementado com alocação dinâmica de memória, manipulação de ponteiros e operações de inserção e remoção em listas encadeadas.
Tecnologias:
- Linguagem C
- Ponteiros
- Alocação dinâmica de memória
- Listas encadeadas

Funcionalidades:
- Emitir senha comum
- Emitir senha VIP
- Chamar próximo cliente
- Exibir fila atual
- Consultar quantidade de clientes na fila
