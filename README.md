# Estrutura de Dados - Lista Encadeada em C

## Descrição

Projeto desenvolvido em C para a disciplina de Estrutura de Dados da FATEC.

O sistema simula uma fila de atendimento com senhas comuns e VIP utilizando listas encadeadas. Permite emitir senhas, 
inserir clientes VIP com prioridade, chamar o próximo atendimento, visualizar a fila e consultar a quantidade de pessoas aguardando.
O projeto foi implementado com alocação dinâmica de memória, manipulação de ponteiros e operações de inserção e remoção em listas encadeadas.

## Funcionalidades

- Emitir senha comum
- Emitir senha VIP
- Chamar próximo cliente
- Exibir fila atual
- Consultar quantidade de pessoas na fila
- Liberação de memória ao encerrar o programa

## Estruturas de Dados Utilizadas

- Lista Encadeada Simples
- Ponteiros
- Alocação dinâmica de memória (`malloc` e `free`)

## Tecnologias

- Linguagem C
- GCC
- Visual Studio Code

## Como Executar
Compilação -> gcc main.c lista.c no.c -o programa

Execução -> ./programa

Conceitos Aplicados
- Manipulação de ponteiros
- Encapsulamento de operações da lista
- Inserção e remoção de nós
- Gerenciamento de memória
- Implementação de filas com prioridade
